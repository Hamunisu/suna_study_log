static

実行中ずっと存在し、どこからでもアクセス可能な不変変数

mutを使用できるが所有権移動が危険なため使用しない方がよい

Lazy

実行中ずっと存在し、呼び出された時に起動する
staticとセットで使うことがある

String::new()と違い、起動時に変数処理を行わないから起動が速くなる
Lazyコードを通過したときに、メモリに枠だけ確保し、呼び出された時にLazyに問い合わせて内容を補完する。次回以降はその箱を参照する

書き方: static 変数名: Lazy＜数値型＞ = Lazy(|| {必要な処理});

*(ディファレンス演算子)

ポインタや参照がさしている実際の値を取り出す

例: *DEV_IDでDEV_IDが指す値を取り出す

env

environmentの略
OSに登録されている情報をプログラムから読み取るための機能

1つであれば自動で読み込まれる
自動で読み込ませるにはdotenvを.tomlに記載する必要あり

HashMap

定型文を作成するもの
素材をまとめたりするのに使える

ある意味jsonの代わりになるものである
結果を表示する場所が必要である
