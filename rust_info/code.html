fn

関数定義の合図

functionの略

main

プログラムのエントリーポイント
必須アイテム

boxの名前
これを忘れるとコンパイルエラーになる！

println!

標準出力（ターミナル等）に文字列を表示させるマクロ

mainの処理（中身）

use

外部のクレートやライブラリなどを呼び出すもの

呼び出せるようにスタンバイさせる
クレート内のものを省略して入力できるようになる

let

変数定義するもの

デフォルトは不変である
let x = 5 と入力する

mut

変数を可変（変更可能）にする

mutable（可変）の略
let mut x = 5と入力する
